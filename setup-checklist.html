BOMexplorer Marketing Site — Setup Checklist

Run these steps once in your terminal before handing over to Claude Code.

Phase 1 — Verify Prerequisites
1
Check Node.js version Required ≥ v18
Astro requires Node 18 or later. You likely have this from the main project (nvm).
node --version
If below v18, run: nvm install 20 && nvm use 20 && nvm alias default 20
2
Check Git Required
Needed for Cloudflare Pages deployment.
git --version
3
Check GitHub CLI Optional but convenient
Lets you create the GitHub repo from the terminal. Install if missing.
gh --version
sudo apt install gh # if not installed
Phase 2 — Scaffold Astro Project
4
Navigate to project directory
cd ~/Documents/code_projects/bomexplorer_marketing
5
Create Astro project in current directory
When prompted: choose "Empty" template, Yes to TypeScript, Yes to install dependencies.
npm create astro@latest -- --template minimal .
The . installs into the current directory. Make sure you're inside bomexplorer_marketing/ first.
6
Add Tailwind CSS
Accept all prompts.
npx astro add tailwind
7
Verify dev server starts
Should open at http://localhost:4321. Ctrl+C to stop.
npm run dev
Phase 3 — Git & GitHub
8
Initialise Git repo & first commit
git init
git add .
git commit -m "chore: scaffold Astro marketing site"
9
Create GitHub repo and push
Using GitHub CLI. Creates a public repo named bomexplorer-marketing.
gh repo create bomexplorer-marketing --public --source=. --remote=origin --push
If you prefer private: replace --public with --private. Cloudflare Pages works with both.
Phase 4 — Cloudflare Pages (do after Claude Code builds the site)
10
Connect repo to Cloudflare Pages
Do this from the Cloudflare dashboard after the site is built.
Cloudflare Pages settings:

Build command: npm run build
Build output directory: dist
Node.js version: 20 (set in Environment Variables as NODE_VERSION = 20)
11
Point bomexplorer.app root domain to Cloudflare Pages
In Cloudflare Pages → Custom Domains → add bomexplorer.app. DNS is managed by Cloudflare so this is automatic.
Once these steps are done, open the project in VS Code and run the Claude Code prompt.
