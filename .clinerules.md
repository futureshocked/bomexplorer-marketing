# BOMexplorer Marketing - Project Rules

## Project Context
- Astro 6 marketing site for BOMexplorer (KiCad BOM pricing/stock tool)
- Dark theme, monospace + sans-serif fonts
- Pages: index, features, pricing, about, contact, faq

## Dev Commands
- `npm run dev` - Start dev server (localhost:4321)
- `npm run build` - Production build

## Style Guidelines
- Dark theme colors: `var(--color-bg)`, `var(--color-surface)`, `var(--color-primary)` (blue), `var(--color-secondary)` (teal)
- Headings: Space Mono (monospace), Body: IBM Plex Sans
- Font sizes, spacing use Tailwind classes
- Components: FeatureCard, PricingCard, FaqItem

## Project Structure
```
src/
  layouts/
    MarketingLayout.astro (wraps all pages)
    BaseLayout.astro (outer shell)
  components/
    Nav.astro, Footer.astro
    FeatureCard.astro, PricingCard.astro, FaqItem.astro
  pages/
    index.astro, features.astro, pricing.astro
    about.astro, contact.astro, faq.astro
